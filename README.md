# MP3-tag-reader-and-editor
A simple and efficient C-based application to read and edit ID3 tags of MP3 files. 
This project allows users to view and modify metadata like title, artist, album, and more directly from the command line.

Features
📖 Read MP3 tag details:
Title
Artist
Album
Year
Genre
Comment
✏️ Edit MP3 tag details:
Modify specific fields (Title, Artist, Album, etc.)
⚡ Fast and lightweight (pure C implementation)
💻 Command-line based too

🛠️ Technologies Used
C Programming
Standard File Handling (fopen, fread, fwrite, fseek)
String Handling (strncpy, strcmp)
